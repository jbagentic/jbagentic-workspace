# JB Agentic Meetup Info

Single source of truth for what JB Agentic is. Use the facts and blurbs below **verbatim** in deliverables so the community is described consistently everywhere — don't paraphrase; edit here and re-sync.

## Facts

- **Why:** Most software work can be AI-automated now — most teams are just getting started. Together, we can accelerate this process.
- **What:** A friendly community in Johor Bahru for people building software with AI. We meet every month to share what's working, honest lessons, and open problems we're still figuring out.
- **Who:** Devs, tech leads, founders, and KL/SG folks back for the weekend — all welcome.
- **When:** 4th Saturday every month.
- **Where:** Johor Bahru (venue varies per event — check the Luma calendar).

## Links

- **Event calendar (Luma):** https://lu.ma/jbagentic (short) — also https://luma.com/jbagentic
- **YouTube:** https://www.youtube.com/@jbagentic (the `@handle` is already the canonical short link)

## Conventions

- Spelling: "JB Agentic" (community) and "JB Agentic Meetup" (the event series). Number events as `JB Agentic Meetup #N`.
- The `#N` is a literal label, not a hashtag — YouTube won't linkify a number-only `#1`. For a clickable series tag use `#JBAgentic`.
- Canonical Luma URL is `https://luma.com/jbagentic`; `https://lu.ma/jbagentic` is the short form (both resolve).

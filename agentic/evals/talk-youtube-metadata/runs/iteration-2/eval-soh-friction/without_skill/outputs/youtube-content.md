## Description

Soh Jun Wei shares his journey building a personal AI-powered expense tracking agent that eliminates the everyday friction of recording expenses. Using Hermes Agent, MCP (Model Context Protocol), and Telegram as the input interface, Jun Wei demonstrates how text messages, voice notes, and receipt photos can be automatically processed and stored — no manual data entry required.

The talk covers the full architecture — from Telegram input through Hermes Agent for fuzzy understanding, to a custom MCP server for deterministic database operations — and explores the key design principle of separating what AI does best (natural language understanding) from what tools do best (validation, calculation, data persistence). Jun Wei also reflects on how AI is shifting the developer role from coder to architect and orchestrator.

Presented at JB Agentic Meetup on 30 May 2026.

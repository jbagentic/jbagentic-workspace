## Description (copy-paste ready)

```
Kowa Jia Liang, a senior software engineer at Payboy (an HRMS SaaS company in Singapore), shares his team's real-world journey from manual coding through ChatGPT and Cursor to their current setup with VS Code + Claude Code.

He walks through the practical tools and techniques that made AI coding work in production: a CLAUDE.md permanent context file, custom Claude skills for changelog generation and automated code auditing, the Superpowers plugin for plan-then-build feature development, and MCP integration with Sentry for debugging. The talk includes a live demo of a code-auditor skill that eliminates 70-80% of redundant PR review comments, and a real project where an MRR report page was shipped in roughly 30 minutes after being delayed for months.

Talk: "From Manual Coding to ChatGPT, Cursor and Claude Code" by Kowa Jia Liang

What you'll learn:
- The evolution from manual coding to ChatGPT to Cursor to Claude Code
- Setting up CLAUDE.md as a permanent context file for your codebase
- Building custom Claude skills for changelog generation and code auditing
- How the Superpowers plugin plans features, then uses parallel agents to build and test
- A real case study: shipping an MRR report page in 30 minutes
- Connecting Sentry via MCP for AI-assisted production debugging
- Honest comparison of Cursor vs Claude Code and why a product-heavy team chose quality over speed
```

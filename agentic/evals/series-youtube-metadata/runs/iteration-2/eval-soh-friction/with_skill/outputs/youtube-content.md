## Description

Manual expense tracking is tedious — what if you could just text "lunch 5.50", send a voice note, or snap a receipt photo and have an AI agent handle the rest? Soh Jun Wei built exactly that: a personal expense agent using Hermes, MCP, and Telegram, and shares what he learned about designing reliable AI applications.

🎤 Talk: "AI for Everyday Friction" by Soh Jun Wei
📍 JB Agentic Meetup #1 — 30 May 2026

What's covered:
• The friction problem with traditional expense-tracking apps
• Building an AI agent that accepts text, voice, and receipt photos via Telegram
• Architecture: Hermes Agent + MCP + SQLite on Oracle Cloud free-tier VPS
• Why MCP gives agents predictable, reusable tool boundaries instead of raw prompting
• Separating fuzzy input (AI's strength) from deterministic actions (tools' strength)
• MCP vs CLI: when each approach fits
• How the developer role shifts from coder to AI orchestrator and architect

Tech stack: Hermes Agent, MCP, Telegram Bot API, SQLite, Next.js, Oracle Cloud, Groq Whisper API, OpenRouter (Qwen 3.5 27B), Gemini Flash Vision

## Description

A senior engineer walks through his team's real journey from manual coding to ChatGPT to Cursor to Claude Code — and shares the custom skills, permanent context files, and workflows that now power their daily development at a Singapore HRMS startup.

🎤 Talk: "From Manual Coding to ChatGPT, Cursor and Claude Code" by Kowa Jia Liang
📍 JB Agentic Meetup #1 — 30 May 2026

What's covered:
• The four stages of AI adoption: manual coding → ChatGPT → Cursor → Claude Code
• Setting up permanent context with CLAUDE.md so the AI already knows your codebase
• Building custom Claude skills for changelog generation and automated PR review
• Using the Superpower plugin to plan and ship a full feature in 30 minutes
• Connecting Claude to Sentry via MCP for automated bug diagnosis
• Why the team chose Claude over Cursor: sub-agents and output quality vs speed
• Live demo of the code-auditor skill eliminating 70–80% of redundant PR comments

Tech stack: Ruby on Rails, Vue.js, Claude Code, Cursor, Sentry, Superpower plugin
